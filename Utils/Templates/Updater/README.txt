Info:

This tool can be used to update the Template definitions int PacketTemplates.xml. 
It will generate and update the file in your Guild Wars® folder.
These files are used to parse and compile the packets for Guild Wars®. 
You only need to update if new Packets were added to Guild Wars®.





Usage:

	-launch GW.exe 
	-inject TemplateUpdater.dll
	-connect to the game server
	-hit insert to update the templates
	-hit delete to eject the updater
