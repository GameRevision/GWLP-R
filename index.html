GuildWars Lan Project : Revived!

This project aims to create a GuildWars server sandbox, with basic gameplay features. For any further reading, see our GitHub page, or see the official GameRevision board on www.gamerevision.com.
We do neither support nor ecourage piracy! We respect the game's creators! All GWLP-R devs are passionate GuildWars players that own at least one account.

Dependencies

We highly depend on Reality:Shard and the Shardlet API, which both can be downloaded here. As Java is the language this whole stuff is written in, you will also need a JRE7 for running GWLPR and JDK7 for development. You will also need our project build tool, maven 3. Any additional dependencies will be downloaded automatically by maven.

Install

This project has not released anything yet. If you want to build and run the sources you'll need JDK7 and maven 3, as said before.
