GWLP-R by GameRevision

See GameRevision.com

This project aims to build a GW private server, or in other words, a GW sandbox. For any further reading, see the official GameRevision board on www.gamerevision.com Note that we don't want to encourage piracy.

Dependencies

Currently: None

Install

You probably need to install a current version of MonoDevelop for the source code, and Dia for any Diagrams.
